# Посібник менеджера

Цей посібник охоплює управління вашим магазином sCommerce через адміністративний інтерфейс.

## Огляд панелі управління

Панель управління sCommerce надає комплексний огляд продуктивності вашого магазину та ключових показників.

### Ключові показники

- **Загальні продажі** - Дохід за обраний період
- **Замовлення** - Кількість розміщених замовлень
- **Товари** - Загальна кількість товарів у каталозі
- **Клієнти** - Кількість зареєстрованих клієнтів
- **Конверсія** - Відсоток відвідувачів, які здійснили покупку
- **Середній чек** - Середня сума замовлення

### Недавня активність

- **Останні замовлення** - Нові замовлення зі статусом та інформацією про клієнта
- **Попередження про низький запас** - Товари з низьким рівнем запасів
- **Нові відгуки** - Свіжі відгуки та рейтинги товарів
- **Системні сповіщення** - Важливі системні повідомлення

## Управління товарами

### Додавання товарів

1. Перейдіть до **Товари** → **Товари**
2. Натисніть **Додати товар**
3. Заповніть інформацію про товар:

#### Базова інформація
- **Назва** - Назва товару (обов'язково)
- **Аліас** - URL-дружня назва (генерується автоматично)
- **Опис** - Детальний опис товару
- **Короткий опис** - Стислий опис для списків
- **Категорія** - Оберіть основну категорію
- **Теги** - Ключові слова для пошуку

#### Ціноутворення
- **Звичайна ціна** - Стандартна ціна продажу
- **Акційна ціна** - Знижена ціна (опційно)
- **Собівартість** - Вартість товару для розрахунку прибутку
- **Валюта** - Валюта ціни

#### Інвентар
- **SKU** - Унікальний код товару
- **Кількість на складі** - Доступна кількість
- **Поріг низького запасу** - Попередження при падінні нижче цього
- **Відстежувати інвентар** - Увімкнути/вимкнути відстеження
- **Дозволити передзамовлення** - Дозволити замовлення при відсутності на складі

#### SEO
- **Мета заголовок** - Заголовок сторінки для пошукових систем
- **Мета опис** - Опис для результатів пошуку
- **Ключові слова** - Релевантні ключові слова
- **Канонічний URL** - Бажаний URL для дублікатів контенту

#### Зображення
- **Основне зображення** - Головне зображення товару
- **Галерея** - Додаткові зображення товару
- **Alt текст** - Альтернативний текст для доступності
- **Порядок зображень** - Перетягніть для зміни порядку

### Варіанти товарів

Створення варіантів товарів для різних розмірів, кольорів тощо:

1. Оберіть **Створити варіант** на сторінці товару
2. Оберіть атрибути варіанту (розмір, колір і т.д.)
3. Встановіть специфічні для варіанту ціни та інвентар
4. Завантажте специфічні для варіанту зображення

### Масові операції

#### Імпорт товарів
1. Перейдіть до **Товари** → **Імпорт/Експорт**
2. Завантажте шаблон Excel
3. Заповніть дані товарів
4. Завантажте заповнений файл
5. Перегляньте та підтвердьте імпорт

#### Експорт товарів
1. Оберіть товари для експорту
2. Оберіть формат експорту (Excel, CSV)
3. Оберіть поля для включення
4. Завантажте файл експорту

#### Масове редагування
1. Оберіть кілька товарів
2. Оберіть **Масове редагування** з дій
3. Оновіть спільні поля (ціна, категорія, статус)
4. Застосуйте зміни до обраних товарів

## Управління категоріями

### Створення категорій

1. Перейдіть до **Товари** → **Категорії**
2. Натисніть **Додати категорію**
3. Заповніть деталі категорії:

#### Базова інформація
- **Назва** - Назва категорії
- **Аліас** - URL-дружня назва
- **Опис** - Опис категорії
- **Батьківська категорія** - Оберіть батьківську для підкатегорій
- **Порядок сортування** - Порядок відображення в навігації

#### SEO
- **Мета заголовок** - SEO-оптимізований заголовок
- **Мета опис** - Опис категорії для пошукових систем
- **Ключові слова** - Релевантні ключові слова
- **Зображення** - Зображення/іконка категорії

#### Налаштування відображення
- **Опубліковано** - Зробити категорію видимою для клієнтів
- **Показати в навігації** - Включити в головне меню
- **Рекомендовано** - Виділити в секціях головної сторінки

## Управління замовленнями

### Обробка замовлень

1. Перейдіть до **Замовлення** → **Замовлення**
2. Натисніть на замовлення для перегляду деталей
3. Оновіть статус замовлення за потреби:

#### Статуси замовлень
- **Очікує** - Замовлення отримано, очікує обробки
- **Обробляється** - Замовлення готується
- **Відправлено** - Замовлення відправлено клієнту
- **Доставлено** - Замовлення отримано клієнтом
- **Скасовано** - Замовлення скасовано
- **Повернено** - Замовлення повернено

### Деталі замовлення

#### Інформація про клієнта
- **Ім'я** - Ім'я клієнта
- **Email** - Контактний email
- **Телефон** - Контактний телефон
- **Адреса** - Адреси доставки та оплати

#### Позиції замовлення
- **Товар** - Назва товару та SKU
- **Кількість** - Кількість замовлено
- **Ціна** - Ціна за одиницю
- **Сума** - Загальна сума рядка

#### Підсумки замовлення
- **Проміжний підсумок** - Сума цін товарів
- **Доставка** - Вартість доставки
- **Податок** - Сума податку
- **Знижка** - Застосовані знижки
- **Разом** - Кінцева сума замовлення

## Управління клієнтами

### Огляд клієнтів

1. Перейдіть до **Клієнти** → **Клієнти**
2. Перегляньте список клієнтів з ключовою інформацією:
   - **Ім'я** - Ім'я клієнта
   - **Email** - Контактний email
   - **Замовлення** - Кількість замовлень
   - **Витрачено** - Загальна сума покупок
   - **Останнє замовлення** - Найостанніша покупка
   - **Статус** - Активний/Неактивний

### Деталі клієнта

#### Особиста інформація
- **Ім'я** - Повне ім'я
- **Email** - Основна адреса email
- **Телефон** - Контактний телефон
- **Дата народження** - День народження (опційно)
- **Дата реєстрації** - Коли створено обліковий запис

#### Адреси
- **Основна адреса доставки** - Головна адреса доставки
- **Основна адреса оплати** - Головна адреса для рахунків
- **Додаткові адреси** - Інші збережені адреси

#### Історія замовлень
- **Всі замовлення** - Повна історія покупок
- **Деталі замовлень** - Інформація про окремі замовлення
- **Повернення** - Історія повернень

1. **Регулярне резервне копіювання** - Часто робіть резервні копії даних
